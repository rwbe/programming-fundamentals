## 📚 Sobre este repositório

Este repositório cobre **Fundamentos de Programação** em módulos numerados — cada pasta é uma etapa.
A estrutura é própria do tema (não um template genérico): a numeração reflete a ordem
recomendada de leitura.

- Pasta `hands-on/` com projetos rodáveis.
- Pasta `materials/` com apostila/cheatsheet em Markdown.
